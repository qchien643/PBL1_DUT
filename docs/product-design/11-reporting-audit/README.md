# 11 - Reporting Audit

## 1. Mục tiêu

Cung cấp báo cáo vận hành cho manager và audit log cho các hành động quan trọng.

## 2. Reporting MVP

| Báo cáo | Nguồn |
| --- | --- |
| Doanh thu ngày | Paid bills |
| Món bán chạy | Served/paid order items |
| Order bị hủy/từ chối | Order history |
| Hiệu suất bếp | Preparation task timestamps |
| Trạng thái bàn | Table/session state |

## 3. Audit scope

Mở bàn, ghép/chuyển bàn, submit/accept/reject order, hủy món, confirm payment, đổi menu, đổi config.
